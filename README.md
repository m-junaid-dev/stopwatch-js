# JavaScript Stopwatch

This is a simple stopwatch built using HTML, CSS, and JavaScript.

I made this project to practice DOM manipulation and working with timers in JavaScript. The stopwatch can start, stop, and reset the time. It also plays a small click sound when buttons are pressed.

## Features

- Start the stopwatch
- Stop the timer
- Reset the timer
- Millisecond time tracking
- Click sound feedback on buttons
- Prevents stopping or resetting before the timer starts

## How it Works

The stopwatch uses JavaScript's `setInterval()` function to update the timer every 10 milliseconds.

The time is stored in three variables:
- minutes
- seconds
- milliseconds

Every interval updates the display on the screen using DOM manipulation.
